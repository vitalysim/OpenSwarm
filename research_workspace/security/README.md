# Security Research Workspace

Tracked files in this directory are templates and profiles for the security research swarm.

Runtime notes, resources, progress files, and scratch artifacts are created under:

- `notes/`
- `resources/`
- `progress/`
- `scratch/`
- `experiments/`
- `outputs/`

Those runtime folders are intentionally separated so generated user content can be ignored later without losing templates.

Put reusable design references, logos, screenshots, and example visuals in `design_assets/`.
